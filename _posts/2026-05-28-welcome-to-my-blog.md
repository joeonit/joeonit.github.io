This is my first post. To add a new one, drop a Markdown file in the
`_posts/` folder named `YEAR-MONTH-DAY-title.md` — for example
`2026-06-01-my-second-post.md`.

## Writing posts

You write in plain Markdown. You get headings, **bold**, *italics*,
[links](https://www.markdownguide.org/), lists:

- one
- two
- three

And code blocks with syntax highlighting:

```python
def greet(name):
    print(f"hello, {name}")
```

> Quotes look like this.

That's it — commit, push, and GitHub rebuilds the site automatically.
